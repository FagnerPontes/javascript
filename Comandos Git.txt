----------Adicionar pasta e arquivos localmente:----------
git pull - Atualização local a partir da nuvem


-----------Adicionar pasta e arquivos na nuvem:-----------

1 - adicionar a pasta e seus conteúdos ao "staging area".
--Pasta individual:
git add nome-da-pasta/
--Todas as pastas:
git add .

2 - Crie um commit para essas mudanças:
git commit -m "Adiciona nova pasta e arquivos"

Atualizar na nuvem:
git push
